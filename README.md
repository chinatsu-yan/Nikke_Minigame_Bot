# **胜利女神：新的希望 \- 小游戏辅助脚本 (Nikke Minigame Auto Bot)**

这是一个基于 Python 和 OpenCV 图像识别构建的小游戏辅助自动化脚本，专为《胜利女神：新的希望》开发。

目前项目已经采用了高度模块化的设计，底层集成了基于 Windows 句柄的内存级后台截图引擎，配合现代化深色 UI 界面，带来极佳的使用体验。

注意：游戏须保持在前台运行。

## **✨ 当前功能状态**

* ✅ **跑酷脚本 (Parkour Bot)**: 已实装。
* 🚧 **打地鼠脚本 (Whack-a-Mole)**: 待开发。

## **🚀 安装与运行**

### **1\. 环境准备**

请确保您的电脑已经安装了 [Python 3.8 或更高版本](https://www.python.org/)，并配置了环境变量。

### **2\. 克隆项目与安装依赖**

建议使用虚拟环境运行此项目，以保持系统环境纯净。

#### 克隆/下载项目到本地后，在项目根目录打开命令行

##### 1\. 创建虚拟环境 (Windows)  
`python -m venv venv`

##### 2\. 激活虚拟环境  
`.\venv\Scripts\activate`

##### 3\. 安装所需依赖包 (推荐使用国内镜像加速)  
`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

### **3\. 启动脚本**

由于涉及到键盘模拟与窗口句柄读取，请在虚拟环境激活的状态下直接运行 `main.py`。

`python main.py`

*(注：代码内置了 UAC 提权逻辑，如果当前不是管理员权限，会自动弹出提权请求)*

## **🎮 使用指南**

1. 打开游戏《胜利女神：新的希望》并进入对应的小游戏准备界面。  
2. 启动本辅助脚本，在下拉菜单中选中游戏窗口名称。  
3. 脚本类型选择已支持的功能（如：跑酷）。  
4. 点击 **“▶️ 启动脚本”**。  
5. 在游戏中点击“开始”，脚本将全自动识别画面并进行操作。如需手动终止，请点击界面上的 **“⏹️ 停止脚本”**。
